# Computer Vision Datasets for Autonomous Driving

## Road segmentation
* [Kitti dataset for road segmentation](http://www.cvlibs.net/datasets/kitti/eval_road.php)

## Multi class segmentation [scene parsing]
* [Kitti dataset](http://www.cvlibs.net/datasets/kitti/eval_semseg.php?benchmark=semantics2015)
* [Camvid dataset](http://web4.cs.ucl.ac.uk/staff/g.brostow/MotionSegRecData/)
* [Cityscapes dataset](https://www.cityscapes-dataset.com/)
* [Berkeley deep drive dataset](http://bdd-data.berkeley.edu/)
* [Mapillary vistas](http://research.mapillary.com/)
* [Daimler Urban Segmentation dataset - Old](http://www.6d-vision.com/scene-labeling)

* [VOC Pascal](http://host.robots.ox.ac.uk/pascal/VOC/)
* [MS Coco](http://cocodataset.org/#home)
